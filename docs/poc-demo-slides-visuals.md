## Screenshot Capture Checklist
- Use desktop width (recommended 1440px+)
- Capture with clean data state (no half-open modal unless intentional)
- Use same browser zoom (100%) for consistency
- Keep sidebar visible for orientation on admin pages
- Use PNG format for clarity
